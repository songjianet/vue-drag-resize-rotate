# 组件是否可以旋转

一个基本组件，<b>`rotatable` </b> prop设置为`true`，因此它可以旋转角度。

```js
<template>
  <div class="view-box">
    <div id="toolbar">
      <input type="checkbox" v-model="rotatable" /> Toggle rotatable
    </div>
    <div class="container">
      <vue-drag-resize-rotate :rotatable.sync="rotatable">
        <p>Basic component that is {{ rotatable ? 'also' : 'not' }} rotatable.</p>
      </vue-drag-resize-rotate>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      rotatable: true
    }
  }
}
</script>

</style>
```
